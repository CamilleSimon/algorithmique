# Exercice 2

## Enoncé

Donner les valeurs de `A`, `B` et `C` après l'exécution de l'algorithme suivant :

```
Variables​
    A, B, C : entiers​
Instructions​
    A ← 5​
    B ← 3​
    C ← A+B​
    A ← 2​
    C ← B-A
```

## Correction

A la fin de l'algorithme, `A` vaut 2, `B` vaut 3 et `C` vaut 1.
